Fireshare

File sharing for hot customers

This is a mock django project which allows users to allow users to upload files, which can subsequently be downloaded by the uploading user, and other users if marked as public.

You can...

  • Register and log-in using a username and password
  • Upload files, allowing the user to select whether the file is ‘public’, or ‘private’
  • View a list of the users previously uploaded files, and files uploaded as ‘public’,
    including upload time/date, and a link to the original upload
